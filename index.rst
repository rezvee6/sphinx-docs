Welcome to the Sample Sphinx + PlantUML docs!
=============================================

This is a minimal example Sphinx project that shows:

- How to integrate **PlantUML** diagrams into Sphinx.
- How to create a **complex diagram**.
- How to enable a **pan/zoom viewer** on the rendered diagram.
